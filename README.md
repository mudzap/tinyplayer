# tinyplayer

Simple CLI music player.

## Objective

This is meant as a simple music player easily integrable into my workflow and desktop usage habits.

## Planned features:

* Basic DSP functions
	- LP, HP filters
	- EQ
	- Fade in, out
	- And so on...
* Playlist handling
* Tag handling
* Control commands
* Wide audio codec support
* Search function
* FFT Visualization

## Non-priority features:

* Streaming capability
* GUI
